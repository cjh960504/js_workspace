1. 변수와 자료형: 컴퓨터에게 일거리를 제공하는 법 제공방법 : 메모리에 올려야 함, cf: 사람은 기억력을 사용 [자료형] 개발자는 컴퓨터가 이해할 수 있는 데이터 형태로 제공 해야하는데, 프로그래밍 언어에서 사용할 수 있는 자료형 3가지밖에 없다. 일거리가 될 수 있는 유형은 결국 3가지밖에 없다.(숫자, 문자, 논리값) 자료형을 미리 결정지어놓는 방식 = 컴파일 언어 ex) int x = 3; 자료형의 판단을 자바스크립트에게 맡기 = 인터프리터 언어 (자료형보고 결정해야하므로 좀 느림..) ex) var x = 3; [변수] 개발자가 데이터를 메모리에 올리는 법? 변수 초기화 ex) var a = 5; // 변수 a를 선언하면서 그 값을 5로 할당 변수란? 수학에서는 변하는 수를 의미하지만, 전산분야에서는 메모리에 생성된 공간! 변수명? 메모리에 생성된 공간을 구분하기 위한 붙인 이름 변수가 메모리에 올라가는 시점? 모든 프로그램은 실행할 때 변수가 메모리에 올라간다. [변수명 선언 시 규칙 : 모든 언어 공통] 1) 숫자로 시작할 수 없다. 2) 특수문자로 시작할 수 없다. 예외) 특수문자로 허용되는 것이 2개($, _) 3) 한글을 사용할 수 없다. 2. 연산자 : 메모리에 올려진 일거리를 어떻게 처리할 지를 결정하는 부호 산술연산자 : + - * / 3. 제오문 : 조건/반복을 처리하여 프로그램을 더 효율적 만들어보기 4. 함수 : 내가 이미 작성한 코드를 재사용하는 방법 5. 배열 : 대령의 데이터를 대상으로 프로그램 짜보기! ------------------------------------------------------------------------------------------------------------------------------------- 6. 객체 : 프로그램으로 현실 세계의 사물, 개념 등을 표현해보자. ex) 총알, 적군, 주인공 ... 아이템.. ------------------------------------------------------------------------------------------------------------------------------------- 7.
